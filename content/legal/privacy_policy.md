## Collection and storage of personal data
### When visiting the website
This website is hosted by **GitHub Pages**, a service provided by GitHub, Inc.  

When you visit the website, GitHub automatically processes technical data in server logs, including:  

- IP address of the requesting device  
- Date and time of access  
- Requested resource (URL, file name)  
- Browser type/version and operating system   
- Referrer URL (previous page)  

This data is processed exclusively for the operation, security, and stability of the hosting service.  
The website operator has no access to personally identifiable server logs.  

Legal basis: Art. 6(1)(f) GDPR (legitimate interest in secure and stable website delivery).  

For more information, see:  
https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement

## Legal basis of processing
Processing is based on Art. 6(1)(f) GDPR (legitimate interest in access control) and Art. 6(1)(b) GDPR (performance of a contract or pre-contractual measures, when login/authentication is used).
 
## Storage and data transfer
Server log data is stored temporarily for a maximum of 14 days and then automatically deleted, unless longer storage is required for security or legal purposes.

When using ORCID or Auth0, your data is transferred to these providers in accordance with their privacy policies.
